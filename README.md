# CobberLearnChemProjects
Foundations of machine learning in chemistry digital lab

AlkaneBoilingPoint:This script plots the boiling point of alkane compounds as a function of the number of carbon atoms in the chain.

PubChemFetcher: This scripts fetches molecular weight, molecular formula, H-bond acceptors, H-bond donors, XlogP values
and SMILES strings for a list of compounds of interest and exports these values to csv file.
